# gift-faq-generator

AEO(Answer Engine Optimization) 친화적 FAQ 생성.

## 규칙
- **4개** FAQ (3개 이하 WARN)
- 실제 검색 쿼리를 질문으로 사용
- 첫 문장에 직접 답변 (Position Zero 최적화)
- 답변 80~150자
- 구체적 상품명 + 가격 포함
- FAQPage 스키마와 호환

## 필수 질문 유형
1. **메인 쿼리**: "{대상} {상황} 선물 {예산}으로 뭐가 좋을까요?"
2. **비교/선택**: "A와 B 중 뭐가 나을까요?" 또는 "실용적인 선물 추천"
3. **상품 디테일**: 에디터 픽 상품의 구체적 장점/차이점
4. **구매 팁**: 어디서 사는 게 좋은지, 포장/배송 관련

## 가독성
- word-break: keep-all 적용
- 인용구는 별도 문장으로 분리
