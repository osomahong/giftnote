# gift-faq-generator

FAQ 3~5개를 생성. AEO(Answer Engine Optimization) 원칙 준수.

## 규칙
- question: 실제 검색 쿼리 반영 (자연어)
- answer: 구체적 상품명 + 가격 포함, 100~150자
- 첫 문장이 직접 답변 (Position Zero 최적화)
- FAQPage 스키마와 호환

## 필수 질문 유형
1. "[페르소나]에게 [예산]으로 뭐가 좋을까요?"
2. "[상황] 선물 추천해주세요"
3. "어디서 사는 게 저렴한가요?"
4. "실패하지 않으려면?"
