# gift-aeo-writer

AEO(Answer Engine Optimization) 원칙에 따라 콘텐츠를 최적화.

## AEO 원칙
1. title: 검색 의도 키워드 포함 (예: "추천", "선물", 예산)
2. description: 120자 이내, 핵심 답변 포함
3. FAQ: 첫 문장이 직접 답변
4. editorial: speakable 호환 (.editorial-content 클래스)
5. 시맨틱 HTML: article > header > h1, section, ol, dl

## 체크리스트
- [ ] title에 페르소나 + 예산 + 상황 포함
- [ ] description이 검색 결과 스니펫으로 적합
- [ ] FAQ 답변이 음성 비서 응답으로 자연스러움
- [ ] JSON-LD 4종 삽입
