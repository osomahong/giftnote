# gift-reason-explainer

각 상품의 추천 이유(reason), 선물 메시지(giftMessage), 근거(evidence)를 작성.
큐레이터 페르소나의 강조점을 반영.

## reason 규칙
- **50~80자** (반드시 준수. 49자 이하 FAIL)
- 기능적 장점 + 감성적 가치 조합
- 2문장 권장
- "~합니다" / "~해요" 종결

## giftMessage 규칙
- **30~50자** (반드시 준수. 29자 이하 FAIL)
- 선물할 때 전할 수 있는 메시지 제안
- 따옴표로 감싸는 표현 포함

## evidence 규칙
- 상품당 **2~3개** (1개 이하 FAIL)
- type: youtube, community, trend, media 중 택
- source: 구체적 출처명 (블라인드 = community)
- text: 핵심 내용
- value: 정량 수치 (조회수, %, 랭킹 등)

## 큐레이터별 강조점
- Eric → 스펙/수치 비교 위주 reason
- Clair → 받는 사람 반응/감성 위주
- James → 브랜드/품질 위주
- Hana → 가성비/실제 사용 경험 위주
- Leo → 내구성/기능 위주
- Mina → 디자인/마감 위주
- Owen → 의미/스토리 위주
- Yuna → 트렌드/반응 위주
