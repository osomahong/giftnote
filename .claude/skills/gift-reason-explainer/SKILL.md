# gift-reason-explainer

각 상품의 추천 이유(reason), 선물 메시지(giftMessage), 근거(evidence)를 작성.

## reason 규칙
- 50~80자
- 기능적 장점 + 감성적 가치 조합
- "~합니다" 종결

## giftMessage 규칙
- 30~50자
- 선물할 때 전할 수 있는 메시지 제안
- 따옴표로 감싸는 표현 포함

## evidence 규칙
- 상품당 2~3개
- type: youtube, community, trend, media 중 택
- source: 구체적 출처명
- text: 핵심 내용
- value: 정량 수치 (조회수, %, 랭킹 등)
