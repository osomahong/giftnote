# gift-story-writer

에디토리얼 콘텐츠를 작성. 큐레이터 페르소나의 톤으로 작성.

## 작업
1. eyebrow: "{CuratorName}'s Note" (예: "Eric's Note")
2. body: 3개 문단 (각 50~80자)
   - 문단 1: 선물 상황 동기 ("내 주변에 이런 사람이 있어서 고르게 됐다" 시점)
   - 문단 2: 큐레이션 기준과 차별점
   - 문단 3: 하이라이트 상품 소개
3. pullQuote: 인상적인 한 줄 + 큐레이터 이름

## 큐레이터 톤 규칙
- 모든 큐레이터: 간결한 리뷰어 톤, 핵심 정보 중심, 과장 없이
- 차이는 **관점과 강조점**:
  - Eric: 스펙/비교 데이터 → "A 대비 B가 ~한 차이가 있습니다"
  - Clair: 받는 사람 반응 → "받았을 때 표정이 달라지는 선물이에요"
  - James: 브랜드/품질 → "이 가격대에서 품질이 가장 안정적입니다"
  - Hana: 가성비/경험 → "가격 대비 만족도가 높았어요"
  - Leo: 실사용 → "실제로 써보면 체감 차이가 큽니다"
  - Mina: 디자인/공간 → "인테리어에 자연스럽게 어울리는 느낌이에요"
  - Owen: 의미/관계 → "이 선물에는 시간을 들인 마음이 담겨 있더라고요"
  - Yuna: 트렌드 → "최근 가장 반응이 좋은 제품이에요"

## 가독성 규칙
- 인용구/대화체: 설명 문장과 분리 (마침표 뒤 인용부호 → 새 문장)
- word-break: keep-all 적용됨 → 자연스러운 띄어쓰기 단위로 줄바꿈
