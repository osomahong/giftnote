# gift-crawl-products

상품 리서치 + 수집 + 검증. orchestrator의 4단계를 담당.

## 현재 사용 플랫폼
- **coupang**: 쿠팡파트너스 API (검색 + 어필리에이트 링크)
- **naver**: 네이버 쇼핑 검색 (향후 API 연동 예정)

## 향후 확장 예정
- kakao (gift.kakao.com) — Playwright 필요
- 29cm, kurly — Playwright 필요
- ssg — SSR 가능

## 상품 리서치 워크플로우

### 1. 후보 리스트업 (7~10개)
주제에 맞는 상품을 넉넉하게 검색:
- 쿠팡 API: 관련 키워드 2~3개로 검색 (예: "여자친구 뷰티 선물", "핸드크림 선물세트")
- 각 검색 결과에서 예산 범위 내 상품 선별
- 이미지 확보 가능 여부 사전 확인 (다나와 pcode 존재 여부)

### 2. 기존 상품 중복 체크
content/collections/*.md의 모든 상품과 비교:
- 같은 상품명 or 같은 브랜드+카테고리 조합 카운트
- **50% 이상 중복** → 상품 교체 필요
- **30% 이하 중복** → 허용 (인기 상품은 여러 큐레이션에 등장 가능)
- 특히 코코도르 디퓨저 같은 범용 상품은 2개 컬렉션까지만

### 3. 최종 5개 선정
7~10개 후보 중 아래 기준으로 5개 선정:
- 이미지 확보 가능한 것만
- 브랜드 다양성 (같은 브랜드 2개 이하)
- 가격 다양성 (예산 범위 내 상/중/하 분포)
- 에디터 픽 1개 선정 (가장 추천하는 상품)

## 쿠팡 상품 검증 (자동)

### Phase 1: 정확한 검색
- 쿠팡파트너스 API로 상품명 검색
- 3중 검증: 브랜드 일치 + 상품명 유사도 30%↑ + 가격 ±50%
- 통과 → affiliateUrl 자동 업데이트

### Phase 2: 대안 검색
- 브랜드 + 카테고리 조합으로 재검색
- 통과 → 업데이트

### Phase 3: 플랫폼 전환
- 쿠팡 미취급 → source를 "naver"로 자동 전환
- 네이버 쇼핑 검색 URL로 설정

## 이미지 수집

### 소스 우선순위
1. **다나와** (`img.danawa.com/prod_img/...`) — pcode 이미지
2. **공식 브랜드몰** — 직접 이미지
3. **네이버 쇼핑** — 검색 결과 이미지

### 사용 금지
- `ads-partners.coupang.com` — 쿠팡 광고 배너 (절대 사용 금지)
- 쿠팡 API의 `productImage`는 상품 이미지로 사용하지 않음

### 이미지 미확보 상품 제외
- 다나와/공식몰/네이버 어디에서도 이미지를 안정적으로 확보할 수 없는 상품은 큐레이션에서 제외
- 해당 상품 대신 이미지 확보 가능한 유사 상품으로 교체

### 접근성 검증 (HTTP HEAD)
- 모든 이미지 URL에 HTTP HEAD → 200 확인
- 404/403 → 다른 pcode 또는 다른 소스에서 재검색
- 다나와 가격비교 종료 상품 → 이미지도 404 → 다른 pcode

### pcode 브랜드 매칭 검증
- 다나와 pcode 페이지 제목에 상품 브랜드명 또는 핵심 키워드 포함 확인
- 불일치 → 다른 pcode 사용
- verify-images.ts 스크립트로 전체 자동 검증

### 교차 검증
- 같은 컬렉션 내 이미지 URL 중복 금지
- 중복 발견 → 해당 상품 이미지 재검색

## evidence 출처 분류 규칙
- youtube: 유튜브 채널/영상
- community: 커뮤니티 (블라인드, 디시인사이드, 클리앙, 오늘의집 등)
- trend: 검색 트렌드, 판매 데이터
- media: 언론/매거진
- **블라인드는 SNS가 아님. type: "community"로 분류**

## 쿠팡 광고 문구
source=coupang 상품에는 ProductCard에서 자동 표시:
"이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다"

## 출력
- products 배열 (Product 타입 준수, image 필드 필수)
- `content/products/{slug}.json` 원본 데이터 저장
- 검증 리포트
