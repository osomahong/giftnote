쿠팡파트너스 API 검색 결과의 상품명/이미지/URL이 의도한 상품과 일치하는지 자동 검증해야 한다.

**Why:** 쿠팡 검색 API가 "블렌더보틀 프로스택"을 검색하면 "서플라이스 쉐이커"를 반환하는 등 엉뚱한 결과가 나올 수 있다. 사용자가 매번 수동으로 링크를 클릭해서 확인하는 것은 불가능.

**How to apply:**
1. 검색 결과의 상품명과 원래 의도한 상품명의 유사도를 자동 검증
2. 브랜드명이 포함되어 있는지 필수 체크
3. 가격이 예산 범위 내인지 체크
4. 검증 실패 시 해당 상품 제외하고 다른 검색어로 재시도
5. 최종 검증 리포트를 출력하여 어떤 상품이 매칭되었는지 투명하게 보여줌
6. **이미지 URL 중복 금지**: 같은 컬렉션 내 2개 이상 상품이 동일 이미지 URL을 공유하면 FAIL → 재검색
7. 모든 문제는 사람에게 묻지 않고 자동으로 대안 탐색 → 해결
