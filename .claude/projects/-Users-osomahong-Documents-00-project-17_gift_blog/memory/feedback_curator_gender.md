여성 큐레이터가 "여자친구 선물"을 추천하면 어색하다. 큐레이터는 선물 주는 사람 입장이므로 대상과의 관계가 자연스러워야 한다.

**Why:** Clair(여성)가 "여자친구 생일 선물"을 추천하면 동성 연인이 아닌 이상 부자연스럽다. 사용자가 콘텐츠 신뢰도에 의문을 가질 수 있다.

**How to apply:**
- "여자친구 선물" → 남성 큐레이터만 (Eric, James, Leo, Owen)
- "남자친구 선물" → 여성 큐레이터만 (Clair, Hana, Mina, Yuna)
- "부모님/직장/친구" → 성별 무관
- 콘텐츠 생성 시 자동 검증, 위반 시 자동 재배정
